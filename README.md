# [2024] 칵테일 바 IFIF
- Spring Boot 기반으로 실제 운영하는 칵테일 바 홈페이지를 만든 개인프로젝트
![IFIF2](https://github.com/user-attachments/assets/597e78c3-624f-4d28-8027-3a91b7473ab1)

# 개요
- 프로젝트 이름 : IFIF
- 프로젝트 수행기간 : 2024.08.01 ~ 2024.09.09
- 개발 언어 및 도구 : JAVA & Spring Boot
- 멤버 : 김민

# 주요 기능
- 로그인,회원가입 (소셜로그인)
- 상품등록,수정
- 장바구니
- 카카오 지도 API
- 포트원 결제
- 이메일인증
- AWS
- JSOUP 크롤링
- 댓글기능
